# 极简记事本 (Minimalist Notepad) - PRD

## 1. 项目简介
**产品名称**：简记 (SimpleNote)  
**核心理念**：极简、快速、无干扰。打开即用，专注于文字记录本身，摒弃一切复杂功能。
**目标用户**：需要快速记录灵感、待办事项或临时文本的用户。

## 2. 功能需求 (MVP - 最小可行性产品)

### 2.1 核心功能
1.  **新建笔记**
    -   首页提供明显的“新建”入口。
    -   点击即进入编辑模式。
2.  **笔记列表**
    -   首页以列表形式展示所有笔记。
    -   展示笔记的第一行作为标题，第二行作为摘要。
    -   显示最后修改时间。
3.  **编辑与查看**
    -   点击列表项进入编辑详情页。
    -   支持纯文本编辑。
    -   **自动保存**：无需手动点击保存按钮，输入即保存 (Debounce 机制)。
4.  **删除笔记**
    -   在列表页或详情页提供删除功能。
    -   删除前进行二次确认。
5.  **数据持久化**
    -   **阶段一 (当前)**：使用浏览器 LocalStorage 存储，保证刷新不丢失，保护隐私，无需登录。
    -   **阶段二 (未来)**：云端同步 (可选)。

### 2.2 UI/UX 交互设计
-   **设计风格**：极简主义 (Minimalism)。大量留白，黑白灰为主色调。
-   **响应式**：完美适配移动端 (Mobile First) 和 桌面端。
-   **暗黑模式**：跟随系统自动切换深色/浅色主题。

## 3. 技术栈建议

-   **构建工具**: Vite (极速启动)
-   **前端框架**: React (生态丰富，组件化) 或 Vue 3
-   **样式库**: Tailwind CSS (快速构建现代 UI)
-   **图标库**: Lucide React / Heroicons
-   **路由**: React Router / Vue Router
-   **状态管理**: 无需复杂库，使用 Context 或简单的 Hooks 即可。

## 4. 开发计划

1.  **初始化项目**: 搭建 Vite + React + Tailwind 环境。
2.  **UI 框架搭建**: 实现基础布局 (Layout)。
3.  **核心逻辑开发**: 实现笔记的 CRUD (增删改查) 及 LocalStorage 存储逻辑。
4.  **界面美化**: 调整排版、字体、间距，适配移动端。
5.  **测试与发布**: 基础功能测试，部署到 GitHub Pages 或 Vercel。
